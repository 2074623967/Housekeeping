# accounting-system

## 模块定位

`accounting-system` 是家政支付平台的账户账务系统，对应第二阶段建设模块。

职责包括：

- 账户主体管理
- 账户开立与销户
- 余额结构管理
- 账户流水
- 冻结与解冻
- 入账规则
- 调账能力
