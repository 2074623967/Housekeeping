# 结算出款系统前端页面与交互说明

## 1. 页面范围

### admin-web

1. 结算批次列表页
2. 服务者结算单列表页
3. 商家结算单列表页
4. 结算单详情页
5. 审核页
6. 出款批次页
7. 出款跟踪页

## 2. 页面职责

### 2.1 结算批次列表页

- 查看批次状态、金额、审核状态、出款状态

### 2.2 结算单列表页

- 查看应结金额、扣减金额、实结金额
- 支持审核、退回、发起出款

### 2.3 结算单详情页

- 展示结算来源、费用项、扣减项和审核记录

### 2.4 出款跟踪页

- 展示批次出款进度和失败明细

## 3. 交互要求

1. 所有金额字段必须高亮展示
2. 审核和出款必须二次确认
3. 失败记录必须支持重试
4. 详情必须可下钻到明细项
