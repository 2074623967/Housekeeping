# 清分清算系统前端页面与交互说明

## 1. 页面范围

### admin-web

1. 清分批次列表页
2. 清分结果列表页
3. 清分规则配置页
4. 费用项配置页
5. 分账明细页
6. 清分任务页

## 2. 页面职责

### 2.1 清分批次列表页

- 查看批次日期、订单数、总金额、状态
- 支持发起清分、重跑清分、查看详情

### 2.2 清分结果列表页

- 查看单笔支付对应的拆分结果
- 展示商家、服务者、平台、渠道费用

### 2.3 清分规则配置页

- 维护清分规则版本
- 支持启停和灰度

### 2.4 费用项配置页

- 维护抽佣、手续费、补贴等费用项

### 2.5 分账明细页

- 展示各收款方分账金额和状态

## 3. 交互要求

1. 列表页支持筛选和分页
2. 规则配置必须版本化
3. 重跑清分必须二次确认
4. 所有结果必须可追溯到支付单
