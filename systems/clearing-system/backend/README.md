# clearing-system backend

## 说明

当前目录用于承载 `clearing-system` 的后端实现。

第一版目标：

1. 清分批次查询与创建
2. 清分结果查询
3. 清分规则维护
4. 费用规则维护
5. 分账明细查询
6. 支付成功事件消费

当前实现阶段：

1. 已补齐 Spring Boot + MyBatis + H2 可运行基线
2. 已补齐清分批次、清分结果、清分规则、费用规则、分账明细、支付成功事件的持久化实现
3. 已补齐 `schema.sql / data.sql / mapper xml / 集成测试`
4. 后续会继续切换到 MySQL，并补齐规则灰度、审计追踪、失败补偿和结算联动
