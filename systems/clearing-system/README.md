# clearing-system

## 模块定位

`clearing-system` 是家政支付平台的清分清算系统，对应第二阶段建设模块。

职责包括：

- 交易清分
- 平台抽佣
- 渠道费用归集
- 服务者应收计算
- 商家应收计算
- 清算批次生成
